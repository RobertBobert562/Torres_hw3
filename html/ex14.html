Robert's Example 14

What is the output of the following?
let a = 2;
a -= 1;
a++;
let b = 8;
b += 2;
const c = a + b * b;
const d = a * b + b;
const e = a * (b + b);
const f = a * b / a;
const g = b / a * a;
console.log(a, b, c, d, e, f, g);

I predict that the program will output A as 2.
B will be 10.
C will be 120.
D will be 20.
E will be 40.
F will be 10.
Lastly G will be 2.

After running the code, I was correct for A & B along with E & F. I was incorrect for C & D & G. this was because of the order of operations.
