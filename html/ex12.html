Variable Scope: What is the output of the following?
let num1 = 0;
{
num1 = 1;
const num2 = 0;
}
console.log(num1);
console.log(num2);

I predict that this program will return num1 as 0 and num2 as 0.

After running the program, I realized that num1 would be 1 since it was changed. I thought num1 would not change because it was altered within the block. my prediction would have been correct if num1 was declared again in the block.
num2 would return an error because it was declared within the block.
