Robert's Example 11

What is the output of the following?
console.log(4 + 5);
console.log("4 + 5");
console.log("4" + "5");
console.log(2 + 3 * 5);
console.log((2 + 3) * 5);
console.log(10 % 3);

I predict that this program will return 9 on the first line.
return 4+5 on the second line.
return 45 on the third line.
return 17 on the fourth line.
return 25 on the fifth line.
return 2 on the sixth line.

After executing the program, I was correct for all except the last one which was actually 1.
Upon further investigation, I found out that this code calculates the remainder of when '10/3' which is 1.
